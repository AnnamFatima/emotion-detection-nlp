# Emotion Detection Project - MCA Final Year

## How to Run the App
1. Install dependencies:
   pip install streamlit scikit-learn pandas joblib

2. Run the web app:
   streamlit run emotion_app.py

## Files Included
- emotion_model.pkl – Trained Logistic Regression model
- tfidf_vectorizer.pkl – TF-IDF vectorizer used for feature extraction
- label_encoder.pkl – Optional: not used in final version
- emotion_dataset.csv – Dataset used for training the model
- main.py – Training script
- emotion_app.py – Streamlit app for emotion detection
